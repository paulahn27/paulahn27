Neumorphism in Adobe XD

Welcome to a tutorial on how to make neumorphic shapes on Adobe XD! Although it may be a difficult process, neumorphism is a great way of adding a cool 3D effect to your next UX/UI design project. In short, the basic concept behind neumorphism is to add two shadows, one with a negative value and one with a positive value, to any sized shape so that it looks like it is either coming out of the screen or going into the screen. This tutorial specifically will teach you how to make a neumorphic circle, but the same process can be applied to any shape that you make in Adobe XD.


How to create a pushed out Neumorphic circle

  1. Press R and hold Shift while dragging the corner of the rectangle to create a simple square.
  2. Set the border radius of the square to 100 to make it a circle.
  3. Under the Appearance option, add a black shadow to your circle with X and Y coordinates of 6 and a Blur of 10.
  4. Duplicate your circle and place the new circle directly on top of the old one.
  5. Add a white shadow to the new circle with X and Y coordinates of -6 and a Blur of 10.
  6. Adjust the opacity of the first circle’s shadow to 20%
  7. Adjust the opacity of the second circle’s shadow to 80%.
  8. If needed, alter the Blur, X and Y coordinates, and color of the circle until you are satisfied with the result.
Note: Different sized shapes will require different amounts of blur and shadow opacity. Play around with these components until you find an amount that looks right to you!


How to create a pushed in Neumorphic circle

  1. Make a circle and add a black border of size 5 under the “Appearance” option
  2. Adjust the object blur of the circle to 5 and name the circle “Dark”
  3. Duplicate the “Dark” circle and name it “Light.”
  4. Position the light circle off to the side and make the border color white and add an object blur of 5.
  5. Duplicate the dark circle again and position it off to the side. Name the circle “Mask.”
  6. Right click the mask circle and under the “Arrange” option click “Bring to Front.”
  7. Position all the elements directly on top of each other.
  8. Move the light circle more towards the top left of the other two circles
  9. Select all elements and right click to mask with shape.
  10. Adjust the Opacity, Object Blur, and Border Size of Color of all the elements until you are satisfied with the result.



Prototyping

  • Link the pushed out neumorphic button on artboard 1 to artboard 2 containing the pushed in neumorphic button.
  • For the Trigger select Tap.
  • For the Action Type select Auto Animate.
  • For the Animation select Snap.
  • Change the duration of the Animation to 0 seconds.

Note: Adobe XD will try to automatically set the animation duration to 0.2 seconds. You must manually delete the .2 in order to make it 0 seconds.

Colors in HTML text

Color codes are ways of representing colors in a format that a computer can interpret and display. They are commonly used in websites and other software applications. There are a variety of formats, including Hex color codes, RGB and HSL values, HTML color names, and others.

Different ways of using text color:

  • Hex color codes
  • HTML color names
  • RGB color values
  • HSL color values

Text color using Hex color codes

The most common way of coloring HTML text is by using hexadecimal color codes (Hex code for short). Simply add a style attribute to the text element you want to color and use the color property with your Hex code. You would want to use a Hex color code for the following reasons:

  1. If you already have a code
  2. If you use a "color picker" program that is a chart of colors with Hex codes attached to each color in the chart

Text color using HTML color names

Another way to color your website's text is by using an HTML color name. The HTML code is similar, just replace the Hex code from the previous step with the name of the color you want to use.

Text color using RGB color values

Insert your RGB values within the rgb() parameter following the color property. When using an RGB value in your website, you can also specify opacity. Instead of rgb() use rgba() – the a is for alpha, the color channel that controls opacity – and after your three color values add a fourth for opacity, on a scale from 0 – 1 (0 for completely transparent, 1 for fully opaque).

Text color using HSL color values

A fourth method for adding color is by using HSL values. Similar to the RGB syntax described above, HSL uses the hsl() prefix, and three values for hue, saturation and lightness. Hue is represented on a scale of 0 – 360, while saturation and lightness are each a percentage between 0% and 100%. Just like RGB, when using HSL you can modify the color opacity right in the color property. Use the hsla() prefix and include a fourth value between 0 and 1 for the level of opacity you need.
